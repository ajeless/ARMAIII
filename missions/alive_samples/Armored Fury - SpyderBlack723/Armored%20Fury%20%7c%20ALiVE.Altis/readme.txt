						Armored Fury | ALiVE
--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

    This mission demonstrates a NATO assault on CSAT held land via an armored charge. Mount your tank and fight alongside invading NATO units in order to complete various tasks and eliminate
the enemy. This mission features custom tasks for C2ISTAR which basically overrides what tasks C2ISTAR will automatically assign to you. The three tasks that C2ISTAR is restricted to in this mission
is "Eliminate Infantry", "Military Objective Assault", and "Destroy Vehicles". The available tasks can be tailored to each mission you make in order to best maximize immersion and enjoyability.





						In order to overwrite the possible tasks in a mission, follow these steps

1. Insert these lines into your init.sqf 

////////////////////////////////////////////////////////////////////////--Tasks--///////////////////////////////////////////////////////////////////////////////////////////////////
waituntil {!isnil "ALIVE_autoGeneratedTasks"};					//Waits until ALiVE initializes the tasking module
ALIVE_autoGeneratedTasks = ["MilAssault","DestroyVehicles","DestroyInfantry"];	// Overrides the tasks that C2ISTAR will automatically assign to players

The third line is what specifies what tasks will be available in the mission. All task names are listed here
ALIVE_autoGeneratedTasks = ["MilAssault","MilDefence","CivAssault","Assassination","TransportInsertion","DestroyVehicles","DestroyInfantry","SabotageBuilding"];


								Example

For example, if you wanted to create a mission where the only two possible auto-assigned tasks created by C2ISTAR were Assasination and SabatogeBuilding, your init.sqf would appear like so

////////////////////////////////////////////////////////////////////////--Tasks--///////////////////////////////////////////////////////////////////////////////////////////////////
waituntil {!isnil "ALIVE_autoGeneratedTasks"};					//Waits until ALiVE initializes the tasking module
ALIVE_autoGeneratedTasks = ["Assassination","SabotageBuilding"];	// Overrides the tasks that C2ISTAR will automatically assign to players
